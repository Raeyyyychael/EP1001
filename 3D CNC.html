3D CNC

3D CNC is to create a mould to so that we are able to use it for moulding and casting. We first have to create our file. It can be done in Fusion360.

Once we have the file prepared, we can then generate the Gcode.
When generating the Gcode we can also use Fusion to generate the Gcode, make sure to use WinCNC as the processor instead of Mach3 as the processor. Alternatively we can use Stepcraft-UCCNC which can be downloaded from the Fusion post library.
